extensões são maneiras de tornar o vscode mais potente 

alt + clique em outras linhas: cursor funciona em todas as linhas, no local que você clicou

ctrl+f -> replace -> mudar as ocorrências em que um determinado nome aparece 

editor word wrap column -> define onde quebra o texto

ctrl+p -> busca de arquivos
ctrl+shift+p -> mostra comandos do vscode 

ctrl k + ctrl s -> comandos de teclado 

emmet(?): função que consiste em vários atalhos/comandos para criar estruturas de código
