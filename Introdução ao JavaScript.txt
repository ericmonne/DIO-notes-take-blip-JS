baseada em protótipos
multiparadigma - permite escolher entre programação funcional, estruturada e POO

Protótipos:
base para uma grande quantidade de estrutura de dados do javascript
conjunto de possibilidades em comum para essas estruturas

Client-Side:
parte da aplicação que tem interação com o usuário

ECMAScript:
conjunto de normas que informa quais funcionalidades são oficialmente lançadas naquela linguagem
